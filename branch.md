# branch
> git branch 명령어 정리

## 개념
- 나무가지라는 의미로 코드의 분기를 의미

## 확인
- `git branch`
- `git log --oneline`

## 생성
- `git branch <branch>`

## 이동
- `git switch <branch>`

## 삭제
- `git branch -d crud`

## 병합
- `git merge <branch>`
